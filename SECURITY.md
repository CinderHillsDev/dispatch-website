# Security Policy

This repository is the static **marketing website** for Dispatch (`dispatchrelay.app`). It contains no
server-side code or user data.

## Reporting a vulnerability

**Please report privately - do not open a public issue.**

- **A vulnerability in the Dispatch product** (the relay itself): report it in the product repository via
  [GitHub Security Advisories](https://github.com/CinderHillsDev/dispatch-platform/security/advisories/new).
- **An issue with this website** (e.g. a content-injection / XSS bug, or a broken/malicious link): report it
  via [this repository's Security Advisories](https://github.com/CinderHillsDev/dispatch-website/security/advisories/new).

Please give us a reasonable window to fix the issue before public disclosure. We'll acknowledge your report
and credit you unless you'd prefer to remain anonymous.
